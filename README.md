### 遊戲說明
你喜歡看電影嗎？歡迎參加電影場景大挑戰！<br>
請在時間內，將3張同部電影場景或元素的卡片配對起來<br>這就來測測你是不是影迷吧？

### 遊戲分級
在限時的100分鐘內，完成
- 8組：A級玩家
- 5-7組：B級玩家
- 3-4組：C級玩家
- 1-2組：D級玩家
